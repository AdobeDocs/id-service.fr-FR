# Configuration d’Opt-in avec des balises {#configuring-opt-in-with-launch}

Simplifiez l’activation de la solution CX Enterprise pour Opt-in à l’aide de balises.

## Configuration d’un scénario d’accord préalable avec des balises {#section-8aa1b58bf8374c938aa8cfdeddbad6ff}

La fonctionnalité [Balises](https://experienceleague.adobe.com/docs/experience-platform/tags/home.html?lang=fr) dans la collecte de données Adobe Experience Platform facilite la configuration d’un scénario d’opt-in avec les solutions Adobe. Vous pouvez simplifier la collecte des consentements de souscription des visiteurs pour les solutions d’entreprise CX en autorisant Analytics, Target, Audience Manager et d’autres solutions d’entreprise CX, ou une partie de celles-ci, à souscrire à votre système de gestion du consentement.

**Configurer l’extension de balise [!UICONTROL Experience Cloud ID Service]**

Si l’extension de balise [!UICONTROL Experience Cloud ID Service] n’est pas encore installée, ouvrez votre propriété, puis cliquez sur *Extensions* > *Catalogue*, survolez l’extension de balise [!UICONTROL Experience Cloud ID Service] avec la souris et cliquez sur *Installer*.

Pour configurer l’extension, ouvrez l’onglet *Extensions* et survolez l’extension avec votre souris. Cliquez ensuite sur *Configurer*.

Pour plus d’informations de référence, consultez la [!UICONTROL Experience Cloud ID Service] extension de balise [présentation](https://experienceleague.adobe.com/docs/experience-platform/tags/extensions/client/id-service/overview.html?lang=fr) (extension qui met en œuvre le service d’identification des visiteurs).
